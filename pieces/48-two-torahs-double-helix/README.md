# Two Torahs, One Sinai — The Double Helix

A 3D double-helix rendered via canvas 2D projection, where two helical ribbons of Hebrew text
intertwine around a shared vertical axis. One ribbon carries words from Bereishit 1:1 (Written
Torah); the other carries phrases from Pirkei Avot 1:1–4 (Oral Torah chain of transmission).

**Theme:** Matan Torah — both Torahs given simultaneously at Sinai (Exodus Rabbah 47:1; Avot 1:1)

**Technique:** canvas 2D — 3D-projected generative typography ribbon, double helix, painter's algorithm

**Palette:**
- Background: deep parchment `#1A1408`
- Written Torah ribbon: warm gold `#D4A830`
- Oral Torah ribbon: deep olive-green `#3A5C1A`
