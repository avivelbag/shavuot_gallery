# Forty-Nine Angles of the Count

*Sefirat HaOmer — pinwheel tiling / aperiodic substitution*

## Overview

A depth-4 pinwheel tiling (Conway–Radin, 1994) generated by recursive subdivision of a 1-2-√5 right triangle. The 49 largest tiles are numbered with Hebrew ordinals א–מ״ט marking each day of the Omer count. The entire tiling rotates continuously at 0.003 rad/frame, making the tiling's unique "all orientations" property visible in real time.

## Technique

Each generation step replaces one 1-2-√5 triangle with 5 similar congruent copies using an affine substitution rule encoded as coordinate transforms. Depth 4 produces 625 leaf triangles drawn as SVG polygons. Tiles are colored by generation depth; the 49 largest by area receive a luminosity boost and Hebrew day labels scaled to each tile's inscribed circle radius.

The rotation is applied via `requestAnimationFrame` updating a CSS `transform: rotate()` on the SVG root element. Frame rate is capped at 60fps via `performance.now()` delta comparison.
