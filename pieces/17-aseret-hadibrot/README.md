# Piece 17 — Aseret HaDibrot

**"The Ten Words"** — Two stone tablets bearing the Hebrew text of the Ten Commandments (Exodus 20:2–14), rendered with a signed-distance field approximation in canvas 2D: each glyph glows with engraved amber fire that breathes on a 12-second pulse cycle.

**Technique:** canvas 2D SDF-style glyph glow, Hebrew typography, stone grain texture

**Theme:** Matan Torah / The Ten Commandments

**Palette:** stone #c8b89a / #a89880, background #1c1612, glow amber #ffcc66–#ff6600, text #f5eed8
