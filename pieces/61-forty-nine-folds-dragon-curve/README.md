# Forty-Nine Folds — The Dragon of the Omer

**Theme:** Sefirat HaOmer — the 49-day count from Pesach to Shavuot (Leviticus 23:15–16)

**Technique:** Canvas 2D — dragon curve via analytical fold-direction sequence, generation-by-generation animation, seven sefirot color bands

## Description

The dragon curve is generated by folding a strip of paper in half 49 times. Each fold adds one new generation of recursive turns; at 14 generations the curve reaches 16,384 segments. The animation draws the curve one generation at a time, pausing ~1 second between generations 1 and 14, with each pair of generations colored by the corresponding sefirah of that week (Chesed through Malkhut). A counter overlay shows the approximate Omer day and the Hebrew sefirah name. The final frame shows all seven colors woven together.

The analytical turn-direction formula `dragonTurn(n) = ((⌊n / lsb(n)⌋ >> 1) & 1) === 0 ? +1 : −1` (where lsb = lowest set bit) computes each turn in O(1), avoiding the need to store 2^49 segments.
