# Before Every Nation — Mondrian Recursive Subdivision

**Theme:** Matan Torah / The Torah Offered to All Nations  
**Technique:** canvas 2D — recursive Mondrian subdivision, sequential dimming, color interpolation

## What this piece does

The canvas is divided recursively into rectangular "kingdoms" using a Mondrian-style
subdivision algorithm biased toward the golden ratio. Each rectangle receives a muted
earth-tone color (ochre, slate, terracotta, olive). Once all ~48 rectangles are formed,
they dim to near-black one by one in sequence — representing each nation's refusal.
One central rectangle remains fully lit, then blooms into warm parchment gold and
displays the Hebrew words נַעֲשֶׂה וְנִשְׁמָע (Exodus 24:7). The animation holds,
then fades and restarts.
