# Forty-Nine Fires — Sefirat HaOmer

**Theme:** Sefirat HaOmer — the 49-day count from Passover to Shavuot (Leviticus 23:15–16)

**Technique:** Canvas 2D — Archimedean spiral, radial gradient glows, time-based animation

## Description

49 glowing dots arranged in a tight Archimedean spiral (7 rotations × 7 dots), each representing one day of the Omer count. The dots ignite sequentially, transitioning in color from ember-orange (day 1) to gold (day 49). When all 49 fires are lit, a silver-white crown at the center blazes to represent Shavuot — the 50th, uncountable day.

Hebrew day labels (יום א׳ through יום מ״ט) appear on hover.
