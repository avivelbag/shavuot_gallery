# Seventy Faces — Poincaré Disk Tiling

**Theme:** Crown of Torah / naaseh v'nishma  
**Technique:** Hyperbolic tiling — Poincaré disk (4,5) reflection group, Möbius rotation animation

## Description

A canvas 2D animation of a hyperbolic tessellation on the Poincaré disk, generated by the (4,5) regular hyperbolic reflection group. Infinitely many congruent tiles fill the unit disk, growing smaller toward the boundary — a direct visual metaphor for the rabbinic teaching that the finite Torah contains infinite meaning (*שבעים פנים לתורה*, BaMidbar Rabbah 13:15).

The central tile displays the letter **ת** (Tav), the final letter of the Torah. Seven palette colors map to the seven weeks of Sefirat HaOmer and the seven biblical species. A slow Möbius rotation (~0.2°/frame, capped at 60fps) makes the boundary tiles shimmer.

## Files

## Mathematical notes

Implemented in pure JavaScript using:
- Complex number Möbius transformations for hyperbolic reflections
- BFS expansion from the central polygon, reflecting across each edge
- Geodesic arcs drawn as circular arcs orthogonal to the unit circle
- Animation: `z → e^(iθ) * z` rotation applied per-frame using `requestAnimationFrame`

The (4,5) group satisfies `4*5 = 20 > 2*(4+5) = 18`, confirming it tiles the hyperbolic plane.
