# The Pattern That Fills the Earth — p6m Wallpaper Group Tiling

**Theme:** Brit Sinai / Matan Torah — Exodus 24:3–7, *naaseh v'nishma*

**Technique:** p6m wallpaper group tiling / canvas 2D symmetry operations

## Description

A static 700×700 canvas piece that generates a repeating pattern using wallpaper group p6m — the most symmetric of the seventeen crystallographic groups that classify all 2D repeating patterns.

The 17 wallpaper groups are the complete mathematical classification of all possible planar symmetry patterns, proved by Evgraf Fedorov in 1891 and confirmed by Pólya in 1924. P6m is the crown of this classification: it has the maximum number of symmetry operations (12 distinct transformations: 6 rotations × 2 reflections), mirroring the twelve pillars Moses erected at Sinai.

The fundamental domain is a 30-60-90 triangle containing a stylized wheat ear (the symbol of Shavuot harvest). It is stamped 12 times per hexagonal tile via p6m symmetry operations (`ctx.save() / ctx.translate() / ctx.rotate() / ctx.scale(-1,1) / ctx.restore()`), then the tile is repeated in a triangular lattice to cover the canvas.

Alternating fundamental domains are coloured field-green (#2A6020) and ivory (#F5EDD8) to make the p6m symmetry visible. Motif strokes are in wheat gold (#D4A830). A 30px header band at the top displays "נַעֲשֶׂה וְנִשְׁמַע" (naaseh v'nishma) in deep green.
