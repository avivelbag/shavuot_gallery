# The Grain That Counts Itself

**Piece 02 — Shavuot Gallery**

The Fibonacci word fractal is a space-filling curve generated by walking the infinite Fibonacci word string (0100101001001010...) and turning left whenever the character is '1'. The result is not merely a pretty curve — it is the same substitution rule that governs phyllotaxis, the packing of seeds in wheat heads, sunflower florets, and pine cones. The fractal is literally the mathematical DNA of the wheat field.

## Connection to Shavuot

Two threads tie this piece to Shavuot:

1. **Wheat and phyllotaxis.** Deuteronomy 8:8 names wheat among the seven species of the Land. The geometry by which wheat seeds pack is Fibonacci phyllotaxis — and the Fibonacci word fractal is that geometry made visible.

2. **Sefirat HaOmer.** The forty-nine day count is structured as 7×7 — seven weeks of seven qualities (the sefirot grid of the Zohar). Fibonacci numbers satisfy F(n) = F(n−1) + F(n−2): each number sums all that came before it, just as each day of the Omer accumulates every preceding day. The animation encodes this: the counter in the canvas corner counts the forty-nine days in real time as the curve is drawn.

## Technical approach

- **Fibonacci word:** Generated on the fly using the Beatty-sequence closed form: `fibWord(n) = floor((n+2)·φ) − floor((n+1)·φ)`, where φ = (1+√5)/2. Returns 1 for '1' (turn) and 2 for '0' (advance only).
- **Depth:** F(22) — 17,711 segments, enough to produce visibly intricate self-similar structure.
- **Canvas:** Full-viewport canvas 2D. Bounding box computed via dry run; step size scaled to fit with 5% margin.
- **Animation:** ~8 segments per frame at 60fps (~37 seconds), then 5-second hold, then loop.
- **Palette:** Wheat gold → amber → field green → sage → gold, using smooth HSL interpolation by segment index.
- **Thumbnail:** Depth-16 walk (987 segments) rendered by `make_thumbnail.py`.
