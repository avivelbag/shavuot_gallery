# Piece 71 — We Will Do and We Will Hear

**Theme:** The Crown of Torah / Naaseh v'Nishma  
**Technique:** Gosper curve / L-system / animated path tracing

A level-4 Gosper curve (flowsnake) generated by L-system expansion and turtle-graphics interpretation. The complete 2401-segment curve is drawn as a faint background; a glowing particle traces the full path over twelve seconds, leaving a gold-to-blue fading trail that reveals the island as it is traveled.

The Gosper curve fills its hexagonal island — you cannot see the whole from inside the path. This mirrors *naaseh v'nishma* (Exodus 24:7): Israel's commitment to do and hear before understanding the whole, discovering the covenant's shape only by following it.

## Technical notes

**L-system rules:** `A → A-B--B+A++AA+B-`, `B → +A-BB--B-A++A+B`, angle 60°.  
Level 4 produces 7⁴ = 2401 segments (2402 points including the start).  
The string is expanded in JS at page load; the turtle walk produces an array of `[x,y]` waypoints that are normalised to the 700×700 canvas with a 28px margin.

**Performance:** The static underlay is pre-rendered once onto an offscreen canvas and stamped each frame via `drawImage`, avoiding re-tracing 2401 segments per frame. The animated trail is drawn segment-by-segment to allow per-segment colour interpolation (gold → blue).

**Color scheme:**
- Canvas background: `#050A14` (near-black deep night)
- Static curve underlay: `#2A3A5A` at 0.5px (barely visible outline)
- Trail: `#C8A020` (gold) at the leading dot fading to `#1A3080` (deep blue) at the tail
- Leading dot: `#F0E8D0` (bright cream), radius 3px, with radial glow gradient
