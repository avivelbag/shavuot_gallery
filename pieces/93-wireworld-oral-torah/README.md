# Moses Received and Transmitted — Wireworld / Oral Torah

**Shavuot theme: Oral Torah / chain of tradition (Pirkei Avot 1:1)**

Wireworld cellular automaton enacting the chain of transmission in Pirkei Avot 1:1. An electron pulse injected at Sinai propagates through a branching wire tree — splitting at each of five generation levels (Moses, Joshua, Elders, Prophets, Great Assembly) — multiplying the signal indefinitely as the Oral Torah multiplied through each generation of teachers and students.

## Technique

Wireworld (Brian Silverman, 1987): each cell has four states — Empty, Conductor, Electron Head, Electron Tail. The transition rules are:
- Empty → Empty
- Electron Head → Electron Tail
- Electron Tail → Conductor
- Conductor → Electron Head if exactly 1 or 2 neighbors are Electron Heads, else stays Conductor

The wire layout is a vertical branching tree. Y-junctions at each generation branch point duplicate the electron signal naturally via the Wireworld rules — no special-casing required. A new pulse is injected into the trunk every 80 animation frames, so signals perpetually traverse the tree.
